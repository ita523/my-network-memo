## Philosophy - Minimal & Sustainable Infrastructure

目指すのは「最高峰」ではなく「必要最低限＋α」。

最高峰の技術は日々目まぐるしく変わる。  
しかし「必要最低限」が変わる速度は、それよりもずっと遅い。  

100%の安全・安心はこの世に存在しない。  
だからこそ、最適な構成は「簡単で」「迅速で」「手間がかからない」こと。  

必要十分で、美しく。  
それがネットワークエンジニアとして、私が目指す構成です。
